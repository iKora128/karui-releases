# Karui（カルイ）

**次世代画像圧縮デスクトップアプリ**

Jpegli（Google製・次世代JPEGエンコーダ）を搭載し、従来のJPEG圧縮と比較して **35%以上の圧縮率向上** を実現。

## ダウンロード

最新版は [Releases](https://github.com/iKora128/karui-releases/releases) からダウンロードできます。

| OS | ファイル |
|----|----------|
| macOS (Apple Silicon) | `Karui_x.x.x_aarch64.dmg` |
| macOS (Intel) | `Karui_x.x.x_x64.dmg` |
| Windows | `Karui_x.x.x_x64-setup.exe` |

## 特徴

- 🖼️ **高圧縮率**: Jpegliエンジンによる次世代圧縮
- ⚡ **高速処理**: SIMD最適化されたリサイズ処理
- 🎯 **シンプルUI**: ドラッグ&ドロップで即座に圧縮
- 📊 **リアルタイムプレビュー**: Before/After比較
